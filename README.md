A specialized mobile-first web application designed to help students near **Jamia Millia Islamia** find suitable Paying Guest (PG) accommodations and compatible roommates.

## ğŸ¯ Problem We Solve

JamiaHub addresses the common challenge students face: finding safe, affordable housing and like-minded flatmates in a university area **without relying on brokers**. We provide transparency, safety, and community-driven insights for the student housing market.

---

## âœ¨ Key Features

### 1. **Smart PG Discovery**
- **Search & Filter**: Find PGs based on specific localities (Okhla Head, Batla House, Zakir Nagar), budget range, and gender preference (Boys, Girls, Co-living)
- **Interactive Map**: See exactly how far a PG is from the university campus or nearest metro station
- **Curated Lists**: Browse "Popular PGs" and "Nearby" sections for quick access to the best options

### 2. **Detailed Property Listings**
- **Rich Media**: High-quality images of rooms, washrooms, and common areas
- **Transparent Info**: Clear breakdown of rent, security deposit, and sharing type (Single/Double)
- **Amenities Checklist**: Icons display available facilities (Wi-Fi, AC, Food/Mess, Laundry, Power Backup)
- **Trust Score**: "Verified by Students" section with crowdsourced ratings for Safety, Cleanliness, and Food Quality

### 3. **Roommate Finder & Matching**
- **Compatibility Scoring**: Advanced matching that analyzes preferences (Smoking, Drinking, Sleep Schedule, Diet) to show "Match %" with potential roommates
- **Detailed Profiles**: View potential flatmates' profiles including course, year of study, and interests
- **Connection Requests**: Send requests to start conversations with compatible roommates

### 4. **Integrated Chat System**
- **In-App Messaging**: Direct messaging with potential roommates or landlords without sharing personal contact info
- **Real-Time Updates**: Message status indicators (Sent, Delivered, Read)
- **Instant Communication**: Discuss rent, rules, and schedules seamlessly

### 5. **Comparison Tool**
- **Side-by-Side View**: Compare up to 3 properties simultaneously
- **Key Metrics**: Analyze Rent, Distance from Campus, Amenities, and more
- **Informed Decisions**: Make better choices with comprehensive comparisons

### 6. **Landlord & Management Tools**
- **Add Listing**: Property owners can easily list PGs with photos and details
- **My Listings Dashboard**: Manage availability and view inquiries
- **Analytics**: Track interest and engagement with listings

### 7. **User Experience (UX)**
- **Mobile-First Design**: Built to feel like a native mobile app (iOS/Android) with bottom navigation and smooth transitions
- **Saved & Shortlist**: Bookmark favorite properties for later review
- **Book Visits**: Schedule physical property inspections directly through the app
- **Touch-Friendly Controls**: Optimized for all device sizes

## ğŸš€ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/PiroMonarch/Jamia_Hub.git
   cd Jamia_Hub
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173` (or the URL shown in your terminal)
